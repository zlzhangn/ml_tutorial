# ML Tutorial

这是一个按机器学习学习路径整理的示例项目，覆盖：
- 基础知识（NumPy / Pandas / 可视化 / 数据预处理 / 特征工程）
- 监督学习（线性回归、逻辑回归、KNN、SVM、朴素贝叶斯、决策树、集成学习）
- 无监督学习（聚类、降维等）

## 目录结构

- `01_Fundamentals/`：机器学习基础与数据处理
- `02_SupervisedLearning/`：监督学习算法示例
- `03_UnsupervisedLearning/`：无监督学习算法示例
- `data/`：示例数据集（如 `heart_disease.csv`、`advertising.csv`）

## 运行环境
- Python 3.10+
